## リポジトリ管理の仕組み

### SSOT連携

全リポジトリの状態は SSOT の `00_SYSTEM/repo-index.yaml` で管理：

```yaml
repos:
  - name: NexusCore
    status: active
    description: マルチLLMルーティングエンジン
    tech: [Python, FastAPI]
    last_updated: 2026-05-31

  - name: atelier-kyo-manager
    status: active
    description: BUYMAセラー管理ツール
    tech: [Python, Playwright]
```

### 自律開発の実践

各リポジトリは Claude Code の自律開発ループで改善：

```
毎時 Cron 実行
    ↓
バックログ Issue を選択
    ↓
Claude Code で自律実装
    ↓
テスト → コミット → SSOT記録
```
